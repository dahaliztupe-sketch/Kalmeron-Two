# محرك الإشراف (Supervisor Engine)

يعد محرك الإشراف البنية التحتية لتنسيق المهام المعقدة في منصة "كلميرون تو". يعتمد على مكتبة `open-multi-agent` لتمكين تعدد الوكلاء.

## البنية
- `src/ai/supervisor/coordinator.ts`: المنطق الرئيسي لتنسيق وتوزيع المهام.
- `src/ai/agents/registry.ts`: السجل المركزي لجميع الوكلاء المتاحين.
- `/app/api/supervisor/route.ts`: نقطة النهاية (API) لاستقبال الأهداف.

## إضافة وكيل جديد
1. قم بإنشاء الوكيل في `src/agents/`.
2. أضف تعريف الوكيل إلى `AgentRegistry` في `src/ai/agents/registry.ts`.
3. سيتعرف المحرك على الوكيل تلقائياً في دورة التنسيق القادمة.

## الاستخدام
استخدم المسار `/api/supervisor` مع إرسال الطلب في جسم الطلب:
```json
{
  "goal": "حلل فكرة منصة تعليمية وقدم خطة عمل أولية"
}
```
