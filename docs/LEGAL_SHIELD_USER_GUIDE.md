# دليل المستخدم: حائط الصد القانوني (Legal Shield)

هذا الدليل يشرح كيفية استخدام ميزات الامتثال والحماية القانونية.

## 1. المرشد القانوني
يمكنك سؤال المرشد القانوني عن:
- ميثاق الشركات الناشئة المصري (2026).
- قانون الحوافز الضريبية (2025).
- قانون حماية البيانات (151 لسنة 2020).

## 2. النماذج القانونية
قم بتعبئة بياناتك لتوليد نماذج مثل NDA، اتفاقية المؤسسين، وغيرها.

## 3. حذف البيانات (الحق في النسيان)
من صفحة الملف الشخصي، يمكنك طلب حذف بياناتك نهائياً.
