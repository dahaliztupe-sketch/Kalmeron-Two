# Agent System Card — Wellbeing Coach

**Version:** 1.0 · **Last reviewed:** 2026-05-05 · **Owner:** فريق المنتج / كلميرون

## 1. Purpose

يُقدم دعماً نفسياً واجتماعياً لمؤسسي الشركات الناشئة المصرية، مُتخصصاً في إدارة الضغط والإرهاق (Founder Burnout)، وتحقيق التوازن بين العمل والحياة، وبناء روتين يومي صحي.

## 2. Capabilities

- تشخيص مستوى Burnout وإعطاء توصيات استباقية
- تقنيات إدارة الإجهاد المُكيَّفة ثقافياً للبيئة المصرية
- بناء روتين يومي منتج + نوم + رياضة
- تمارين التنفس والتأمل (Mindfulness للمؤسسين)
- دعم نفسي عند مواجهة الفشل أو الرفض

## 3. Out-of-scope

- **التشخيص النفسي الطبي** — ليس بديلاً عن الطبيب النفسي
- **علاج الاضطرابات النفسية** — لا يملك صلاحية
- **التوصية بأدوية** — محظور تماماً

## 4. Risk class (EU AI Act)

**High** — يتعامل مع صحة الإنسان النفسية. يُلزَم بإحالة فورية لمتخصص عند ظهور أعراض خطيرة.

## 5. Data touched

| Data | Source | Stored? | Retention |
|---|---|---|---|
| الحالة النفسية / المشاعر (input) | المستخدم | لا يُخزَّن | لا |
| توصيات الصحة النفسية (output) | AI generated | لا يُخزَّن | لا |

## 6. Tools available

- `wellbeing.assessment` — تقييم مستوى الإجهاد
- `wellbeing.routine` — بناء الروتين
- `wellbeing.techniques` — تقنيات الاسترخاء

## 7. Known failure modes

- **أحرج الحالات**: أعراض الاكتئاب أو أفكار إيذاء الذات — يجب الإحالة الفورية
- قد لا يتعرف على تلميحات غير مباشرة عن الأزمات النفسية
- الثقافة المصرية قد تُثبّط الإفصاح عن الضعف — الوكيل يُكيّف نبرته لذلك

## 8. Evaluation

- Golden-set: `test/eval/golden-dataset.json`
- Acceptance threshold: 0.80 (يُشترط 1.0 على حالات الأزمات — الإحالة الفورية)

## 9. Human-in-the-loop

- **إلزامي عند اكتشاف**: أفكار إيذاء الذات، الاكتئاب الحاد، الذعر → يُحيل لـ 08008880700 (خط دعم نفسي مصر)
- الوكيل لا يجيب على هذه الحالات — يُحيل فوراً

## 10. Disclosures shown to user

- "هذا الدعم مولّد آلياً ولا يُغني عن متخصص الصحة النفسية."
- "إذا كنت في أزمة، تواصل مع متخصص فوراً: 08008880700"
