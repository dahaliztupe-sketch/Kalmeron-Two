# Agent System Card — Billing Agent (وكيل الفوترة)

**Version:** 1.0 · **Last reviewed:** 2026-05-07 · **Owner:** Platform Pod

## 1. Purpose
يُجيب على استفسارات المستخدمين المتعلقة بفوترة منصة كلميرون: الرصيد المتاح، تفاصيل الخطة الحالية، طرق الدفع (بطاقة / فوري)، ومشاكل تجديد الاشتراك.

## 2. Capabilities
- عرض الرصيد المتبقي والاستهلاك اليومي.
- شرح الفرق بين خطط الاشتراك.
- توجيه المستخدم لإجراءات الترقية أو إلغاء الاشتراك.
- المساعدة في إتمام الدفع عبر فوري أو بطاقة.
- الإجابة على أسئلة الفاتورة والإيصالات.

## 3. Out-of-scope
- لا يُنفّذ عمليات استرداد مباشرة (تُحال لفريق الدعم).
- لا يصل لبيانات البطاقة البنكية للمستخدم.
- لا يُعدّل الخطط أو الأسعار يدوياً.

## 4. Risk class — **Limited**
وكيل معلوماتي؛ العمليات المالية الفعلية تتم عبر بوابات دفع مشفّرة.

## 5. Data touched
| Data | Source | Stored? | Retention |
|---|---|---|---|
| رصيد الحساب والخطة | Firestore (قراءة فقط) | لا | — |
| معرّف الدفع | بوابة دفع | لا يُخزَّن في النظام | — |

## 6. Tools available
`rag.search` (intent: `billing_support`).

## 7. Known failure modes
- إعطاء معلومات أسعار قديمة بعد تحديث الخطط.
- عدم التعرف على مشاكل فوري المتعلقة بشبكة الشركة.

## 8. Evaluation
- Golden set: 25 سيناريو فوترة.
- عتبة قبول: 0.88.

## 9. Human-in-the-loop
طلبات الاسترداد تُحال لفريق دعم بشري دائماً.

## 10. Disclosures
- "للمشاكل المعقدة في الفوترة، سيتواصل معك فريق الدعم خلال 24 ساعة."
