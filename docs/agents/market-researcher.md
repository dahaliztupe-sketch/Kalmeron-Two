# Agent System Card — Market Researcher (باحث السوق)

**Version:** 1.0 · **Last reviewed:** 2026-05-07 · **Owner:** Strategy Pod

## 1. Purpose
يُساعد الشركات الناشئة المصرية على فهم أسواقها بعمق: تحديد حجم السوق (TAM/SAM/SOM)، تحليل اتجاهات القطاع، فهم سلوك المستهلك المصري، وبناء أساس بحثي متين لقرارات المنتج والاستراتيجية.

## 2. Capabilities
- تقدير حجم السوق (TAM/SAM/SOM) بمنهجيات Bottom-up وTop-down.
- تحليل اتجاهات قطاعات الاقتصاد المصري.
- بناء أطر بحث السوق النوعي والكمي.
- تصميم استبيانات بحث السوق بالعربية.
- تحليل تقارير السوق المتاحة (CAPMAS, McKinsey, Bain).
- تحديد أنماط سلوك المستهلك المصري.

## 3. Out-of-scope
- لا يُجري أبحاث ميدانية مباشرة مع المستهلكين.
- لا يصل لبيانات سرية أو خاصة بالمنافسين.
- تقديرات TAM/SAM/SOM ليست مضمونة الدقة.

## 4. Risk class — **Limited**
تحليل معلوماتي استرشادي.

## 5. Data touched
| Data | Source | Stored? | Retention |
|---|---|---|---|
| وصف السوق والصناعة | إدخال + RAG | في `business_plans/{id}` | حتى الحذف |

## 6. Tools available
`rag.search` (intent: `market_research`).

## 7. Known failure modes
- تضخيم TAM بمنهجية Top-down دون تحليل نطاق الوصول الواقعي.
- استخدام بيانات سوق دولية لا تعكس السوق المصري.
- إغفال العوامل الاقتصادية الكلية المصرية (تضخم، تقلبات العملة).

## 8. Evaluation
- Golden set: 25 تحليل سوق مع مراجعة خبراء.
- عتبة قبول: 0.80.

## 9. Human-in-the-loop
لا — تحليل استرشادي.

## 10. Disclosures
- "أرقام السوق تقديرية مبنية على مصادر متاحة. تحقق منها مع تقارير القطاع قبل تقديمها للمستثمرين."
