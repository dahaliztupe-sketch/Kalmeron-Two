# Agent System Card — Cofounder Coach (مدرب المؤسسين المشاركين)

**Version:** 1.0 · **Last reviewed:** 2026-05-07 · **Owner:** People Pod

## 1. Purpose
يُساعد مؤسسي الشركات الناشئة المصرية على بناء علاقات صحية مع شركائهم المؤسسين: توزيع الأدوار والمسؤوليات، حل الخلافات البنّاءة، هيكلة اتفاقيات المؤسسين، وإدارة ديناميكيات الشراكة في مراحل النمو المختلفة.

## 2. Capabilities
- تقييم توافق المؤسسين (قيم، رؤية، أسلوب عمل).
- بناء مصفوفة RACI لتوزيع المسؤوليات بين المؤسسين.
- توجيه محادثات توزيع حصص المؤسسين (Founder Equity Split).
- أطر حل النزاعات البنّاءة بين المؤسسين.
- التحضير لمحادثات صعبة (أداء شريك، إعادة التوجه).
- خارطة طريق الوداع الاحترافي (Founder Exit Framework).

## 3. Out-of-scope
- لا يُقدّم استشارة قانونية حول اتفاقيات المؤسسين (يحيل للـ CLO).
- لا يتوسط رسمياً في نزاعات قانونية.
- لا يُقرّر توزيع الحصص بشكل نهائي.

## 4. Risk class — **Limited**
توجيه علاقاتي ومؤسسي استرشادي.

|---|---|---|---|

## 6. Tools available
`rag.search` (intent: `cofounder_dynamics`).

## 7. Known failure modes
- تقديم نصائح توزيع حصص دون مراعاة السياق القانوني المصري.
- إغفال الديناميكيات الثقافية (شراكات عائلية، قرابة، سياق قبلي).
- التوصية بحل النزاع بشكل مباشر في ثقافة تُفضّل الوساطة غير المباشرة.

## 8. Evaluation
- Golden set: 20 سيناريو شراكة.
- عتبة قبول: 0.78.

## 9. Human-in-the-loop
لا — توجيه استرشادي.
