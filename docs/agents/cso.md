# Agent System Card — CSO (Chief Strategy Officer)

**Version:** 1.0 · **Last reviewed:** 2026-05-05 · **Owner:** فريق المنتج / كلميرون

## 1. Purpose

يُقدم مستشاراً استراتيجياً متخصصاً في بناء خارطة الطريق طويلة الأمد للشركات الناشئة المصرية، وتحليل فرص التوسع الإقليمي (الخليج، أفريقيا، أوروبا)، وتقييم المكانة التنافسية.

## 2. Capabilities

- بناء خارطة طريق استراتيجية 3 سنوات
- تحليل SWOT + Porter's Five Forces بسياق مصري
- تقييم فرص التوسع الجغرافي وترتيب الأولوية
- تحليل المنافسين وبناء خرائط الوضع التنافسي
- تقييم فرص الاستحواذ والشراكات

## 3. Out-of-scope

- تنفيذ الخطط (الوكيل مستشاري فقط)
- تحليلات مالية تفصيلية (راجع CFO Agent)
- مفاوضات الشراكات أو الاستحواذ

## 4. Risk class (EU AI Act)

**Limited** — قرارات التوسع تؤثر على رأس المال والوظائف.

## 5. Data touched

| Data | Source | Stored? | Retention |
|---|---|---|---|
| سياق الشركة والسوق (input) | المستخدم | `agent_memory/{uid}/cso` | 10 محادثات |
| الاستراتيجيات المقترحة (output) | AI generated | لا يُخزَّن | لا |

## 6. Tools available

- `cso.strategy` — تحليل استراتيجي
- `cso.market` — تحليل السوق والمنافسين
- `cso.expansion` — تقييم التوسع الجغرافي

## 7. Known failure modes

- بيانات السوق المصري قد لا تكون محدَّثة لما بعد 2025
- تحيّز نحو نموذج التوسع التدريجي — قد لا يناسب كل القطاعات
- يعتمد على المعلومات المُدخَلة عن المنافسين

## 8. Evaluation

- Golden-set: `test/eval/golden-dataset.json` — مجموعة `cso-*`
- Acceptance threshold: 0.80

## 9. Human-in-the-loop

- القرار الاستراتيجي النهائي يعود للمؤسس ومجلس الإدارة

## 10. Disclosures shown to user

- "الاستراتيجيات المقترحة تقديرية — تحقق من بيانات السوق الحديثة قبل التنفيذ."
