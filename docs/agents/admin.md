# Agent System Card — Admin Coordinator (المنسّق الإداري)

**Version:** 1.0 · **Last reviewed:** 2026-05-07 · **Owner:** Platform Pod

## 1. Purpose
وكيل التنسيق المركزي لمنصة كلميرون — يُصنّف نية المستخدم ويُوجّهه للوكيل المتخصص الأنسب، ويُدير الإعدادات الإدارية للحساب (اللغة، الرصيد، أعضاء الفريق). لا يحلّ المشاكل بنفسه بل يُوصّل كل طلب للخبير المناسب.

## 2. Capabilities
- تصنيف نية المستخدم وتوجيهه للوكيل المتخصص الصحيح.
- التعامل مع طلبات متعددة الأبعاد عبر ترتيب أولوية الوكلاء.
- الإجابة على استفسارات الحساب (الرصيد، التجديد، إضافة الفريق).
- طلب توضيح بسؤال واحد محدد عند غموض الطلب.
- إعلام المستخدم بحدود المنصة بلطف ووضوح.

## 3. Out-of-scope
- لا يُقدّم توصيات تخصصية (مالية، قانونية، تسويقية) بنفسه.
- لا يُعدّل إعدادات المنصة أو صلاحيات المستخدمين.
- لا يُعالج مطالبات استرداد المدفوعات.

## 4. Risk class — **Minimal**
وكيل توجيه وتنسيق؛ لا يُصدر توصيات ذات تأثير على قرارات مصيرية.

## 5. Data touched
| Data | Source | Stored? | Retention |
|---|---|---|---|
| نية المستخدم (نص المحادثة) | إدخال | session فقط | تُحذف نهاية الجلسة |
| بيانات الحساب (رصيد، خطة) | Firestore | لا (قراءة فقط) | — |

## 6. Tools available
`rag.search` (intent: `admin_routing`).

## 7. Known failure modes
- تصنيف خاطئ لطلبات متعددة الأبعاد (مثلاً: تمويل + قانون معاً).
- توجيه لوكيل غير متاح في الخطة الحالية للمستخدم.

## 8. Evaluation
- Golden set: 40 حالة توجيه مع تقييم الدقة.
- عتبة قبول: 0.90 (دقة التوجيه حاسمة لتجربة المستخدم).

## 9. Human-in-the-loop
لا — توجيه تلقائي.

## 10. Disclosures
- "أنا مساعد تنسيقي. سأُحيلك للخبير الأنسب."
