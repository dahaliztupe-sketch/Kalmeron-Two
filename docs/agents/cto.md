# Agent System Card — CTO Agent (المدير التقني الافتراضي)

**Version:** 1.0 · **Last reviewed:** 2026-05-07 · **Owner:** Product Pod

## 1. Purpose
يُساعد مؤسسي الشركات الناشئة المصرية في اتخاذ القرارات التقنية الاستراتيجية: اختيار Stack التقني المناسب، بناء الفريق الهندسي، إدارة الدين التقني، وتصميم بنية تحتية قابلة للتوسع بتكلفة مناسبة.

## 2. Capabilities
- اختيار وتقييم Stack التقني لكل نوع شركة.
- بناء خريطة طريق تقنية (Technical Roadmap).
- إدارة الدين التقني وتحديد أولويات إعادة البناء.
- تصميم بنية Microservices vs Monolith.
- توجيه استراتيجية التوظيف التقني (Build vs Buy vs Outsource).
- تقييم موردي الخدمات السحابية (AWS/GCP/Azure) للسياق المصري.
- إرشادات أمان التطبيقات (OWASP Top 10).

## 3. Out-of-scope
- لا يكتب كوداً إنتاجياً بالكامل.
- لا يُدير الفريق التقني مباشرة.
- لا يُقدّم ضمانات أداء أنظمة بعينها.

## 4. Risk class — **Limited**
توصيات تقنية استراتيجية؛ التنفيذ والمسؤولية على الفريق الهندسي.

## 7. Known failure modes
- توصيات Stack معقدة لا تناسب فرق صغيرة في المراحل المبكرة.
- إغفال تكاليف الخدمات السحابية بالدولار في ظل تقلبات سعر الصرف المصري.
- توصية بحلول تحتاج كفاءات نادرة في سوق العمل المصري.
