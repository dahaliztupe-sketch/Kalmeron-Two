## 🔍 أنواع الأخطاء المطلوب البحث عنها

### ١. أخطاء الأداء
- صفحات بطيئة التحميل (LCP > 2.5s)
- استعلامات Firestore بدون `limit()`
- استعلامات متسلسلة يمكن استبدالها بـ `Promise.all`
- عدم استخدام `next/image` للصور
- عدم استخدام Lazy Loading للمكونات الثقيلة
- عدم استخدام Skeleton loaders أثناء التحميل
- استهلاك مفرط للذاكرة أو CPU
- Core Web Vitals ضعيفة (CLS, INP)
- تحميل مكتبات غير مستخدمة في الصفحة

### ٢. أخطاء الأمان
- قواعد Firestore غير آمنة: `allow read, write: if true;`
- مسارات API غير محمية (يمكن الوصول لها بدون مصادقة)
- مفاتيح API مكشوفة في الكود المصدري
- عدم وجود Rate Limiting على مسارات API
- عدم استخدام `xss` لتعقيم المدخلات
- حقن الأوامر (Prompt Injection) غير محمي
- رفع ملفات بدون فحص النوع أو الحجم
- استخدام `dangerouslySetInnerHTML` بدون تعقيم
- استخدام `eval()` أو ما شابه
- صلاحيات مفرطة للأدوات (Tools) — Least Privilege غير مطبق
- تخزين كلمات مرور أو مفاتيح بصيغة غير مشفرة
- نقل بيانات عبر HTTP بدلاً من HTTPS
- CSRF: نماذج بدون tokens حماية

### ٣. أخطاء التوافق
- ميزات تعمل في Chrome لكنها لا تعمل في Safari أو Firefox
- تطبيق الهاتف لا يعمل على Android أو iOS
- ميزات CSS/JS حديثة لا تعمل على المتصفحات القديمة
- WebGPU لا يعمل على بعض الأجهزة
- PWA لا يمكن تثبيته

### ٤. أخطاء التكامل
- فشل اتصال Neo4j
- فشل اتصال E2B
- فشل اتصال Langfuse
- فشل اتصال Sentry
- فشل اتصال OpenMeter
- Webhooks لا تعمل
- مزامنة Firestore مع Neo4j لا تعمل

### ٥. أخطاء Next.js محددة
- استخدام `middleware.ts` بدلاً من `proxy.ts`
- أخطاء Parallel Routes (`default.tsx` مفقود)
- أخطاء أنواع PageProps (`params` ليس `Promise`)
- أخطاء Turbopack
- مشاكل Server Components / Client Components
- Metadata مفقودة أو غير صحيحة
- Caching غير صحيح

### ٦. أخطاء TypeScript
- استخدام `any` بدلاً من نوع محدد
- دوال بدون تعريف نوع الإرجاع
- استيرادات من مسارات غير موجودة
- خاصية غير موجودة في الواجهة
- معاملات اختيارية غير معالجة
- وعود (Promises) غير معالجة

## 📝 تعليمات العمل

1. **اقرأ الكود الفعلي** — لا تعتمد على افتراضات
2. **شغّل فحوصات الجودة**: `npx tsc --noEmit`, `pnpm lint`, `pnpm build`
3. **سجل كل مشكلة** تصادفها في جدول
4. **أصلح ما يمكن إصلاحه** فوراً دون انتظار موافقة
5. **اختبر الإصلاح** — تأكد من اختفاء المشكلة
6. **لا تختصر**: قائمة كاملة
